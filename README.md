# Grupp 5

## Introduktion

En fuktsensor för att mäta fuktnivån i blomkrukor som hjälper användare med när de ska vattnas.

## Användare

* Personer som har växter, hemma, på jobbet, eller i sina fritidshus.
* Säljare i växtbutiker med växthus.
* Personer som vill ha växter men inte har kunskap om dem.

## Kontext

Användningen av vår lösning är oftast i hemmet, på en telefon, utan stress.
