# Grupp 5

## Introduktion

En fuktsensor för att mäta fuktnivån i blomkrukor som hjälper användare med när de ska vattnas.

## Användare

Vuxna männsikor som har problem att komma ihåg att vattna sina blommor

## Kontext

Användningen av vår lösning är oftast i hemmet, på en telefon, utan stress.
